# QA-Knowledge-Base

> Q&amp;A LLM Online Knowledge Base

闽高联问答机器人官方文档向量库

## 仓库规范

1. 本仓库目录结构应与 [闽高联](https://www.kdocs.cn/team/2396183976) 在线文档目录结构保持一致，便于管理人员同步
2. 在每个文档开头应使用 `>` 语法标记本文档的同步时间与在线文档地址，时间应使用 `yyyy-MM-dd` 格式。示例
    ```markdown
    > 在线文档: https://www.kdocs.cn/l/cj9vPT3172P5
    > 最新同步时间: 2024-10-12
    ```
